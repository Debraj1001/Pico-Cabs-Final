# Pico-cabs | Admin Operations Dashboard

A premium, European-style SaaS dashboard designed for real-time ride-hailing operations and fleet management. Built with technical precision and a minimalist aesthetic.

## 🚀 Technical Stack
- **Framework**: React 18 + Vite + TypeScript
- **Styling**: Tailwind CSS (European Minimalist Theme)
- **Animations**: GSAP (Technical Precision feel)
- **Icons**: Lucide React
- **Charts**: Recharts
- **Navigation**: React Router 6

## 🛠️ Key Features
1. **Fleet Overview (Dashboard)**: KPI grid with real-time revenue, ride tickers, and high-level system metrics.
2. **Fleet Personnel (Driver Management)**: High-density data tables for fleet compliance, monitoring, and personnel operations.
3. **Asset Registry (Vehicles)**: Complete visibility into vehicle statuses, maintenance schedules, and assignments.
4. **Operations (Trips)**: Real-time tracking of active trips and historical logs.
5. **Emergency Hub (SOS Center)**: Mission-critical "War Room" for emergency response with tactical map interfaces.
6. **Financials**: Detailed ledger for payouts, revenue tracking, and financial analytics.
7. **Premium Aesthetic**: Deep navy/black base, minimal borders, and professional typography to ensure a distraction-free environment.

## 🏁 Getting Started

### Prerequisites
Make sure you have Node.js and npm installed on your system.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Helix-1716/Pico-cabs.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Pico-cabs
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Run the development server:
   ```bash
   npm run dev
   ```

5. Open your browser and navigate to the URL provided by Vite (usually `http://localhost:5173`).

### Production Build
To create a production-ready build, run:
```bash
npm run build
```
